## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## ğŸ¨ Customization

### Personal Information

Update the following files with your personal information:

1. **Hero Section** (`src/components/Hero.js`):
   - Update name, title, and description
   - Add your social media links

2. **About Section** (`src/components/About.js`):
   - Update personal description
   - Modify statistics and achievements

3. **Skills Section** (`src/components/Skills.js`):
   - Add your technical skills
   - Update proficiency levels
   - Modify skill categories

4. **Experience Section** (`src/components/Experience.js`):
   - Add your work experience
   - Update education information
   - Modify achievements and technologies

5. **Projects Section** (`src/components/Projects.js`):
   - Add your projects
   - Update project descriptions and links
   - Modify technologies used

6. **Contact Section** (`src/components/Contact.js`):
   - Update contact information
   - Add your social media links

### Styling

- **Colors**: Update CSS custom properties in `src/App.css`
- **Fonts**: Change fonts in `public/index.html` and CSS files
- **Layout**: Modify grid layouts and spacing in component CSS files

### Images

Replace placeholder images with your actual photos:
- Add your profile picture to the Hero section
- Update project images in the Projects section
